# Module 7: Holland RIASEC Implementation

## Обзор
Модуль 7 теперь реализует **Типологию профессиональных интересов по Голланду (RIASEC)** вместо "Карьерных якорей Шейна".

## Система Holland RIASEC

### 6 типов личности
1. **R - Realistic (Реалістичний)** - Практик/Делатель
   - Работа с инструментами, машинами, природой
   - Конкретные практические задачи

2. **I - Investigative (Дослідницький)** - Мыслитель/Исследователь
   - Анализ, научное мышление
   - Решение сложных абстрактных задач

3. **A - Artistic (Артистичний)** - Творец
   - Самовыражение, творчество
   - Нестандартные решения

4. **S - Social (Соціальний)** - Помощник
   - Помощь людям, обучение
   - Эмпатия, забота о других

5. **E - Enterprising (Підприємницький)** - Лидер
   - Влияние, убеждение, управление
   - Достижение целей

6. **C - Conventional (Конвенційний)** - Организатор
   - Порядок, структура, четкие правила
   - Работа с данными

## Трехбуквенный код
Результат теста - **код из 3 букв**, например:
- **RAI** - Realistic-Artistic-Investigative (практичный творческий исследователь)
- **SAI** - Social-Artistic-Investigative (помогающий творческий мыслитель)
- **RCE** - Realistic-Conventional-Enterprising (практичный организованный лидер)

### Интерпретация кода
- **Первая буква**: основной тип личности (наиболее выражен)
- **Вторая буква**: дополнительный тип (второй по значимости)
- **Третья буква**: третий по значимости тип

## Гексагон Голланда
Типы, расположенные рядом на шестиугольнике, более похожи:
- R ↔ I (Практик ↔ Мыслитель)
- I ↔ A (Мыслитель ↔ Творец)
- A ↔ S (Творец ↔ Помощник)
- S ↔ E (Помощник ↔ Лидер)
- E ↔ C (Лидер ↔ Организатор)
- C ↔ R (Организатор ↔ Практик)

Противоположные типы (R ↔ S, I ↔ E, A ↔ C) более различаются.
